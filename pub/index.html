This is a WebSDR receiver, located in Heppen - Limburg - Belgium - JO21oc - 70m ASL.

It is operated by Bas ON5HB and Marian ON3MS, e-mail ==> Bas and Marian <==
Beware when we are transmitting ourselves the reception can be disturbed.

More information about the WebSDR project can be found on http://www.websdr.org.

NOTE 1 you need both Java and JavaScript enabled for this page to work properly.
If you don't hear anything, probably Java is disabled or its version is too old (i.e., pre-1.4.2).

NOTE 2 If you have an Android- or Apple-powered tablet or mobile phone, try this one: ==> Mobile version <==
